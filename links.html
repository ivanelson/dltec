Criando Links e Ancores em HTML

Link Externo

Dizemos que um link eh externo quando ele aponta para uma pagina que estah ffora do seu site.
Por exemplo, podemos fazer um link para o google e, nesse caso, ele serah um link externo.

Clique aqui para ir para o google.

Nos links -- tag "a" -- temos o atributo target que define o local onde a pagina serah aberta, conforme o link aponta.

Abrir link do google na mesma janela.
Abrir link do google em uma nova janela.

OBS: caso nao configurarmos o target, o valor default=_self serah mantido, onde a pagina vai abrir na mesma janela.

Link Interno

Dizemos que um link eh interno quando ele aponta para uma pagina que esta dentro do site atual.
Por exemplo, podemos fazer um link para a pagina "texto.html" usando o href e o destino sendo a propria pagina.

Ancoras

Ancoras nada mais sao do que links que apontam para algum lugar da pagina que voce estah.
Um exemplo muito comum de encontrarmos eh o "voltar ao topo".

Para criarmos uma ancora, precisamos utilizar um atributo global "id" visto anteriormente nesse capitulo.

A logica eh bem simples, basta adicionar um id em algum elemento de sua pagina e depois criar um link que "aponte" para esse id.

Veja como eh facil... primeiro vamos la no titulo h1 da nossa pagina e criamos um id="topo". Veja como ficaria abaixo... e em seguida faca isso no seu arquivo.

<h1 id="topo">Criando Links e Ancoras em HTML</h1>

Agora que ja temos um elemento com um id definido eh so criar um link que aponte para esse id... veja como seria:

<a href="#topo">Voltar ao Topo</a1>
