Criando Listas em HTML

Listas nao ordenadas

As listas nao ordenadas sao muito comuns em Website, a tag para sua criacao eh a "ul". Alem do mais cada elemento da lista eh criado com a tag "li"

Vamos a um exemplo. Isso eh uma lista nao ordenada:
  • Item 1
  • Item 2
  • Item 3

Alem do mais podemos ter uma lista dentro de outra lista... basta adicionar um outra tag "ul" e seus itens "li" dentro da lista principal, veja:

  • Item 1
    • Item 1
    • Item 2
    • Item 3
  • Item 2
  • Item 3

Listas ordenadas

Eh uma lista ordenada o padrao eh que cada item tenha um numero sequencia, iniciando 1. Uma lista ordenada eh criada com a tag "ol" e cada item da lista com a tag "li".

Exemplo de lista ordenada:

  1. Introducao 1
  2. HTML 5
  3. CSS 3

Lista de descricoes

Outro tipo muito legal de listas eh a de descricoes... cada item da lista tera uma descricao. Utilizamos a tag "dl" para criar a lista, em seguida, cada item com a tag "dt" e sua descricao com a tag "dd".

Exemplo de lista de descricao:

Lista Nao ordenada
Neste tipo de lista, cada item eh mostrado com simbolo de Bullet
Lista ordenada
Neste tipo de lista, cada item eh mostrado com simbolo de Bullet
Lista de descricao
Neste tipo de lista, cada item tem sua descricao
